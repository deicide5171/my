1. 법인사업자 vs 개인사업자

차이점1 : 투자유치의 가능성
법인 : 주식 및 회사채 발행 가능, M&A 투자유치 가능성 좋음
개인 : 투자를 통한 성장 어려움

차이점2 : 세부담의 관점
법인 : 10~22%
개인 : 6~38%
* 수입이 적을수록 개인이 세부담이 적다
2160만원(과세표준기준) = 순이익 // 연간소득금액이 2160만원 넘으면 개인이 세부담이 크다. // 연간소득금액이 커지면 법인으로 전환하여 세부담을 관리하고 투자를 받아 확장하는 형태로 진행

차이점3 : 법적 실체가 다르다
법인 : 창업자와 법인이 별개 책임
개인 : 본인이 직접

차이점4 : 설립의 용이성
법인 : 복잡(여러가지 서류 작성. 2~3일 걸림)
개인 : 쉬움(사업자등록만 하면됨)

차이점5 : 운영 및 관리
법인 : 회사 대표라도 돈을 마음대로 사용 불가 // 외부감사 존재(큰 회사 좋은 회사라고 생각하면됨) // Compliance 이사회, 주주총회 등등 // 스타트업, 벤처기업
개인 : 자금운영이 편함 // 일정규모가 커지면 성실신고대상자가 되어 감사 // Compliance 없음 // ex> 동네슈퍼

- 부가가치세가 면제!(프리랜서 = 면세사업자) 업체에서 3.3% 원천징수함. 프리랜서는 소득신고만.

- 발주처에서 프로젝트 진행하려면 사업자등록이 필요하다고 요구(경비처리를 위해서 요구함)
- 적격증빙 : 세금계산서, 신용카드 매출전표, 원천징수 영수증
- 원천징수 영수증만 있어도 경비처리 가능!!!!! 사업자 등록 필요없어도 됩니다.
